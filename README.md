# About
This repository contains vital analytical and summarizing files for the project designated to the research of grammatical gender in perception of Russian-German bilingual speakers. The purpose and the reslts of the research are described in the "Отчёт" file.
The experiment was conducted using the [PciBex]([url](https://farm.pcibex.net/)) software; the stimuli are provided here after undergoing the preprocessing, the procedure of which is available at the PciBex website.
